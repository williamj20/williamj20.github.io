Projects

Employee Management System

This program was made using Java and the NetBeans IDE. This program serves to allow a user to store employee information, using a hash table. Functions include adding, editing, removing, and listing all employees and their personal information. The program also has the ability to save and load employee information from to and from an external text file. I've also documented a user manual describing all functions and features of the system, allowing users to use the program to its fullest extent. Here's the manual

Tic Tac Toe

Coded using Python, this tic tac toe game has two game modes - a player vs. player, and a player vs. computer. This game uses an algorithm that allows the CPU to analyze the state of the board and produce the best move possible.

Catch the Block

Coded using Python, this game was made using the module Pygame. The goal of this game is to use the arrow keys to catch a falling block. Holding down a key allows you to accelerate to ensure you are able to catch up to the speed of the falling block. This game was the first time I ever developed anything like a game, and I plan on improving all aspects of the game in the future.

Experiences

105 Royal Canadian Army Cadets (2014-19)

I was an army cadet for five years, where I achieved the rank of Warrant Officer. As a cadet, I was able to learn how to transition from being a follower to a leader and in my final year, I was delegated to the role of being a star level instructor, where I taught a class of 12 of cadets weekly. Some of my responsibilities included commanding a battery and organizing and supervising events and activities, including two-hour night-navigation activities during field-training-exercises.

As a cadet, I also participated in some extracurricular activities within the corps. In my first two years, I was a member of the marching band. In my final two years, I was a part of the marksmanship team, competing in competitions against other cadet Corps from all around Ontario. I've volunteered for fundraisers for the cadet program and for Remembrance day, and I've participated (and performed in the marching band!) in several Remembrance day parades.

Student Activity Council (2018-19)

In my final year of high school, I tried out and made the student activity council as an upper council member. It was here that I was able to meet fantastic new people and work together as a team with the common goal of promoting school spirit and organizing fun activities. Upper council members were responsible for making advertisement posters, promoting events, and assisting in the organization, setup, and supervision of events. Notable events included school spirit days, semi-formal, and a carnival.
